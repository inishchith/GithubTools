============

A Basic information parser for Github :

### Tools
  | File | Description |
  | :--- | :----- |
  |  `GithubIssues.py` | Gives Total Issues , Total PR's and much more |
  | `GithubUser.py` | Gives User Information |
  | `GithubRepo.py` | Gives Repositories Information |
